
# LauraArrayList

Esse projeto em java foi criado com o objetivo de colocar em prática os conceitos de estrutura de dados com o ArrayList em Java. Onde criei meu proprio ArrayList com os métodos:

- add(Object o)
- duplicateCapacity()
- add( int index, Object element )
- remove( int index )
- remove( Object element )
- set(int index, Object element)
- get( int index )
- indexOf(Object o)
- toArray()
-  contains(Object o)
- toString()
